# 14

## Default Up & Down

To run default `compose.yml file`

```shell
docker compose up
```

```shell
docker compose down
```

## Emoji API

### Up

To run specific `yml` file 

```shell
docker compose -f compose-emoji-api.ymls
```

### Down

```shell
docker compose down
```

## Wordpress

### Up

```shell
 docker compose -f wordpress.yml up
 ```

### Down

 ```shell
 docker compose -f wordpress.yml down
 ```
